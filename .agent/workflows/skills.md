**IDE or AI agent specific skills** (e.g., Gemini, Github Copilot, etc.) can be found **in the AI agent's specific directories** (e.g., `.gemini/skills/`, `.github/skills/`, etc.).  
**This workspace's specific skills** (i.e., agent prompts & workflows) can be found **in `docs/agents/skills/`**.

If you are looking for skill definitions, documentation, or implementations, please refer to those directories.

**Tip!** This directory is provided for discoverability and compatibility with tools or agents that expect ALL available skills to be here.
